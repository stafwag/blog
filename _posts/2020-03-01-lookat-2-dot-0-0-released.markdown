Lookat 2.0.0 is the latest stable release of Lookat/Bekijk the userfriendly file browser/viewer.

### ChangeLog

 * utf8 support
 * default color scheme has been updated
 * improved error handeling
 * Macos 10.7+ support
<!--more-->
